Q9. Create an ordered list of the most common text formatting tags in HTML. Within each list item, use an unordered list to list the specific use cases and best practices for that tag.

Most common text formatting tags in HTML

  1. Bold tag
    • The < b > tag- It does not hold any extra importance other than making text bold.
    • Bold texts are used to highlight words that are important.
  2. Italic tag
    • The < i > tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.
    • The < i > tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.
  3. Underline tag
    • The < u > tag represents some text that is unarticulated and styled differently from normal text.
    • The content inside is typically displayed with an underline.
    • Tip: Avoid using the < u > element where it could be confused for a hyperlink!
  4. Strong tag
    • This text is important!
    • The < strong > tag is used to define text with strong importance. The content inside is typically displayed in bold.
    • Tip: Use the < b > tag to specify bold text without any extra importance!
  5. Emphasized text.
    • The < em > tag is used to define emphasized text. The content inside is typically displayed in italic.
    • A screen reader will pronounce the words in with an emphasis, using verbal stress.
  6. Marked text.
    • The < mark > tag defines text that should be marked or highlighted.
    • Highlight parts of a text.
  7. Deleted text.
    • The < del > tag defines text that has been deleted from a document.
    • Browsers will usually strike a line through deleted text.
  8. Subscript tag
    • The < sub > tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font.
    • Subscript text can be used for chemical formulas, like H2O.
  9. Superscript text.
    • The < sup > tag defines superscript text.
    • Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.
    • Superscript text can be used for footnotes.
